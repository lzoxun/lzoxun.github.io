web前端存储

发表于 2018-12-30

cookies

一直跟在http请求头里
只有4k大小
主域名设置的cookies 子域名也会携带
Cookie中设置了”HttpOnly”属性 JavaScript将无法读取到cookies信息

userdata （只有IE支持）

localstorage（本地存储）

优劣

localstorage && sessionstorage
localstorage 如果没有手动删除永久存在
sessionstorage 重新打开浏览器关闭页面就清除了,刷新是不会消失的
一个域名有5M左右大小
与服务端无关
不和子域名共享

api

存储数组或json需先序列化

储存图片

先转为base64 ，储存这个base64就可以了

application cache(离线缓存)

indexedDB(本地数据库)

类似以前的web sql，是一种在浏览器中持久储存结构化数据的数据库，并未web应用提供了丰富的查询能力
移动端支持比较少

在一个域名下可创建多个DB，一个DB下可以创建多个表
