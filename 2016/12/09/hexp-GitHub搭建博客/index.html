hexp+GitHub搭建博客

发表于 2016-12-09

1、基本介绍

==> 首先安装node与git

==> 进入hexo.io 按照给的提升全局安装hexo、创建hexo项目、npm install 等

  • npm install hexo-cli -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo server

==> 进入 _config.yml 文件设置一下博客配置 如 title 等

配置完成之后需要 $ hexo generate 一下重新生成文件

==> \scaffolds\ 下得文件表示不同的网页 文章是基于页面新建的

==> \source\_postsm 下存放的是写的文章

写文章可通过 hexo new "文章名" || 直接新建md文件  
hexo new 的文章默认以 `\scaffolds\post.md` 文件为模板
hexo new page "页面名"  新建一个页面里面有一个默认的index.md 文件
设置 post_asset_folder: TRUE 之后新建时同时生产相应的资源文件夹(直接访问)

==> public 本地部署所在文件夹

==> 部署时会将项目中 \source\img\* 的图片部署到使用环境的img文件夹中

2、主题切换

==> 主题晚上随意下载、存放在 目录\themes 下就可以使用
==> 主题切换: 将_config.yml 下 theme 得值设为需要展示的主题就好了
==> 主题中也有 _config.yml 文件配置针对于该主题的一些信息

3、部署到git上

==> 首先安装hexo-deployer-git

==> 其次修改配置(_config.yml 的最后)

==> 一件部署之后就能直接访问
